# 日語 N5 → N3 完全特訓

直接用瀏覽器打開 `index.html` 即可使用，不需要網路、不需要安裝任何東西。
進度（XP、星等、錯題本、字卡記憶排程）自動存在瀏覽器 localStorage。

> 💡 改了 `data/` 內容後若瀏覽器沒更新，按 **Ctrl+F5（強制重新整理）** 清快取即可。

## 模組

1. **文法課程（28 課）**：每課=「為什麼學這個・怎麼用」框＋文法卡片（例句可朗讀）＋練習題（選擇/填空/重組/配對），答錯進錯題本
2. **單字・漢字字卡（SRS）**：Anki 式間隔重複——四鍵評分（重來/困難/普通/簡單），系統自動排程下次復習時間。**每天回來清「待復習」是關鍵**。卡片可自動朗讀（按 R 重聽）
3. **🔊 發音・重音教室**：日語高低重音（pitch accent）入門，每個詞有重音曲線圖＋朗讀，含「橋／箸／端」「雨／飴」等辨義例
4. **🎧 聽力測驗**：每個牌組可做「聽發音→選意思」
5. **學習計畫表**：約 8 週的逐日進度建議（進度自己抓）

內容量：N4 文法 16 課＋N3 文法 12 課（共 555 題）；字卡共 **2414 張**＝N4 單字 600＋N4 漢字 168＋N3 單字 1450（達 N3 所需量約 ≥80%）＋N3 漢字 100＋生活實用片語 96。

## 朗讀 / 聽力說明（重要）

朗讀用瀏覽器內建的語音合成（Web Speech API），**免費、離線可用、不需金鑰**。
- 建議用 **Edge 或 Chrome** 開啟，發音最自然。
- 若沒聲音或非日語發音：Windows 設定 →「時間與語言」→「語言和區域」→ 新增「日本語」→ 安裝語音套件，重開瀏覽器即可。
- 重音曲線是教學用的標準東京腔；實際 TTS 語音的重音由系統語音決定，當作「聽感參考」即可。

## 使用方式

- 文法：先讀文法卡再練習，拿 ★★（75%）以上再前進
- 字卡：點牌組「學習」開始；可先按「瀏覽單字表」整批看過一遍
- 填空題與字卡**可直接打羅馬字**（`tabete`→たべて），不需日文輸入法
- 字卡鍵盤操作：空白鍵翻面、1〜4 評分
- 每日新卡上限預設 20/牌組（首頁可調）

## SRS 評分怎麼按

- **重來**：完全想不起來 → 1 分鐘後重現
- **困難**：想很久才想起來 → 短間隔
- **普通**：想起來了 → 正常間隔（1天→2.5天→6天…越來越長）
- **簡單**：秒答 → 跳更長間隔

## 如何修改、擴充內容

所有內容在 `data/`，引擎（index.html）不用動。新增檔案後在 index.html 加一行 `<script src>`。

### 文法課（COURSE_DAYS）

```js
window.COURSE_DAYS.push({
  id:'day29',                 // 唯一 id
  stage:'N3 文法（第 17～28 課）',  // 首頁分組用
  title:'Day 29 · 標題', sub:'副標', goals:['目標'],
  lessons:[ {h:'卡片標題', b:'HTML 內容'} ],
  questions:[ /* 四種題型，見下 */ ]
});
```

題型：
```js
{t:'mc', q:'題目', c:['正解','錯1','錯2'], x:'解說'}       // c[0]必為正解，顯示時自動打亂
{t:'input', q:'題目', a:['平假名','漢字'], x:'解說'}        // a=可接受答案
{t:'scramble', q:'中文', tiles:['正','確','順序'], extra:['干擾塊'], x:'解說'}
{t:'match', q:'題目', pairs:[['左','右'],...], x:'解說'}
```

lessons 的 b 可用樣式：`<table>` 變化表、`<div class="ex">例句</div>`、`<div class="tip">提醒</div>`。

### 字卡牌組（VOCAB_DECKS）

```js
window.VOCAB_DECKS.push({
  id:'vn3-09',            // 唯一 id（改了 id 會重置該牌組的記憶進度）
  title:'N3 單字 9｜主題', level:'N3',
  cards:[ {w:'単語', k:'たんご', m:'中文意思', e:'短例句（選填）'} ]
});
```

> N3 單字目前有 8 包（400字）、漢字 5 包（100字）。要補到全量（單字約1800、漢字370），
> 直接把這個 README 和一個現成 data 檔丟給 AI，請它「依同格式產出 N3 單字包 9〜12」即可。

### 文法課的「為什麼學這個」框（why）

每個 day 物件可加 `why` 欄位，會渲染成課程最上方的醒目方塊：

```js
why:{b:`為什麼要學（可用 HTML）`, use:`實際會用到的場景例句`}
```

### 隨堂小卡（翻面例題）

每個 day 物件可加 `lessonCards`，key 是 lesson 的索引（第 0 個 lesson＝0），value 是該概念下方的翻面卡陣列（正面題目、點擊翻面看解答）。每個概念建議 2 張以上：

```js
lessonCards:{
  0:[ {q:'正面：想想看的題目', a:'背面：解答＋詳解（可用 HTML <b> 等）'},
      {q:'…', a:'…'} ],
  1:[ … ]
}
```

### 概念錯題統計（每題對應到一個核心概念）

每課可加 `cptMap`：一個與 `questions` 等長的陣列，第 N 個數字＝第 N 題所屬的**概念（lesson 索引，0 起算）**。例如：

```js
cptMap:[0,0,0, 1,1,1, 2,2, ...],  // 對應 ① ② ③… 各 lesson
questions:[ ... ]                  // 長度需與 cptMap 一致
```

有了 cptMap，系統會統計每個概念的作答/答錯次數：
- 在該課教材頁，每個概念標題旁顯示「✗ 答錯 N 次（共作答 M）」或「✓ 目前全對」，讓你知道是哪個概念不熟。
- 做完一次該課測驗後，結果頁會顯示「📊 這次錯題集中在這些概念」的長條分布。

> 復習錯題仍用既有的「📕 錯題本」功能；此處只做**概念層級的統計回饋**。目前 cptMap 已套用於第 1～10 課。

### 發音・重音教室（data/pronunciation.js）

`window.PRON_LESSON = { intro:'HTML', sections:[ {h, b, examples:[...]} ] }`。
每個 example 可帶 `mora:['は','し','が']` 和 `pitch:'HLL'`（H=高拍、L=低拍，長度需等於 mora）就會自動畫出重音曲線；`say` 指定朗讀文字。

### 學習計畫表（data/plan.js）

`{sec:'階段標題'}` 是分段列、`{d:'第N天', g:'文法', v:'單字漢字'}` 是日程列。

### 其他可調

- 星等門檻：index.html 的 `starsOf()`（90/75/60%）
- SRS 參數：`LEARN_STEPS`（學習步驟）、`applyGrade()`（間隔倍率）
- 重置進度：首頁「🗑 重設進度」
